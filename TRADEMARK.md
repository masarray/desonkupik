# Trademark and Brand Guidelines

The names **DeSonKuPik** and **SonKuPik**, the official logo, app icon, artwork, visual identity, official presets, and product branding are SonKuPik brand assets.

The source code license does not automatically grant permission to use the official brand identity in a way that suggests endorsement, official status, or ownership by someone else.

## Allowed uses

You may use the names DeSonKuPik and SonKuPik to:

- Refer to the official project.
- Link to the official repository.
- Describe compatibility or a fork's origin.
- Write tutorials, reviews, documentation, or news about the project.

## Not allowed without written permission

Please do not:

- Publish a fork as if it is the official DeSonKuPik release.
- Use the official icon or logo for an unrelated product.
- Resell or redistribute modified builds using the official branding in a misleading way.
- Claim sponsorship, partnership, certification, or endorsement by SonKuPik without permission.

## Forks

Forks are welcome under the source code license. Public forks should use a clearly different name and branding when distributing modified binaries to avoid user confusion.
