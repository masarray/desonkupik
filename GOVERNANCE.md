# Governance

DeSonKuPik is maintained by SonKuPik.

## Maintainer responsibilities

Maintainers are responsible for:

- Project direction.
- Release decisions.
- Security triage.
- Reviewing pull requests.
- Protecting project branding.
- Keeping documentation accurate.

## Decision making

Project decisions should prioritize:

1. User safety.
2. Audio quality and reliability.
3. Beginner-friendly workflow.
4. Cross-platform maintainability.
5. Clear documentation.

## Contributions

Contributions are welcome through issues and pull requests. The maintainers may decline changes that do not match the product direction or quality standard.

## Branding

Official branding is managed by SonKuPik. See `TRADEMARK.md`.
