# Support

## Community support

Use GitHub Issues for reproducible bugs and feature requests.

Use GitHub Discussions if enabled for questions, ideas, and general feedback.

## Before opening an issue

Please include:

- App version.
- Operating system.
- Whether you used web or desktop.
- Steps to reproduce.
- What you expected.
- What actually happened.
- Screenshots or logs if useful.
