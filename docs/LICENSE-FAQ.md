# License FAQ

## What license does the source code use?

The source code is marked as GPL-3.0-or-later.

## Are the app name and logo included in the same permission?

No. Source code permission and brand usage are separate topics. See `../TRADEMARK.md`.

## Can I fork the project?

Yes, subject to the source license. If you distribute a modified app, use clear naming so users do not confuse it with the official DeSonKuPik release.

## Can I use the official app for work?

Yes. Normal app use is allowed under the project terms.

## Can I make a white-label product?

For white-label, OEM, official branding, or enterprise partnership, contact SonKuPik. See `../COMMERCIAL-SERVICES.md`.

## Is this legal advice?

No. This is a plain-language project FAQ.
