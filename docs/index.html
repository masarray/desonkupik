FAQ

Questions creators usually ask.

Is DeSonKuPik free?

DeSonKuPik is published as a public open-source project with desktop release artifacts on GitHub.

Does it work offline?

The desktop app is designed for local processing and native file open/export workflow.

What platforms are supported?

The release workflow prepares builds for Windows, macOS, and Linux. The web app can run on static hosting.

Who is it for?

Content creators, reviewers, musicians, editors, and audio enthusiasts who want a fast and beautiful mastering workflow.
