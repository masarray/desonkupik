FAQ for beginners

Clear answers before you install.

What is DeSonKuPik?

DeSonKuPik is an audio mastering application. In simple words, it helps make your audio sound more polished, clearer, and more ready for publishing.

What is mastering?

Mastering is the final audio finishing step. It usually improves loudness, tonal balance, clarity, stereo feel, and peak safety so the audio feels more finished.

Is this a music maker or beat maker?

No. DeSonKuPik does not create songs from scratch. It is for improving an existing audio file that you already have.

Do I need to understand EQ, compressor, and limiter?

No. The default chain is designed as a starting point for beginners. Advanced users can still understand the chain, but basic users can simply open, listen, and export.

Will it always make every file better?

No tool can guarantee that every file becomes perfect. A clean recording will usually get better results. A badly clipped, noisy, or distorted recording may still have limitations.

Does it repair broken audio?

It is not a restoration tool. It can help with tone and finishing, but it is not meant to remove heavy noise, fix clipped recordings, or separate vocals from music.

Is my audio uploaded to the internet?

The desktop app is designed for local file workflow. Normal desktop use opens and exports files on your computer.

Can I use it for YouTube videos?

Yes. You can use it to improve voiceover, music demos, review audio, or exported video audio before publishing.

Can I use it for podcasts?

Yes. It can help make speech sound more controlled and polished, but recording quality and microphone technique still matter.

Which file should I export?

Use WAV when you want high quality for editing or archiving. Use MP3 when you need a smaller file for sharing.

Why are Windows and macOS showing security warnings?

Open-source builds are often unsigned. Windows SmartScreen or macOS Gatekeeper may warn because the app is not yet code-signed by a paid certificate.

Can I use it commercially?

Check the repository license and business-use information first. For company redistribution, OEM, or closed-source commercial use, contact SonKuPik.

FAQ untuk user awam

Jawaban jelas sebelum install.

DeSonKuPik itu aplikasi apa?

DeSonKuPik adalah aplikasi mastering audio. Sederhananya, aplikasi ini membantu audio terdengar lebih rapi, lebih jelas, dan lebih siap dipublikasikan.

Mastering itu maksudnya apa?

Mastering adalah tahap finishing audio. Biasanya tujuannya membuat loudness lebih pas, suara lebih seimbang, detail lebih jelas, stereo lebih enak, dan output lebih aman dari clipping.

Apakah ini aplikasi pembuat musik?

Bukan. DeSonKuPik tidak membuat lagu dari nol. Aplikasi ini dipakai untuk memperbaiki atau memoles file audio yang sudah ada.

Apakah saya harus paham EQ, compressor, limiter?

Tidak harus. Preset/chain bawaan dibuat sebagai titik awal untuk user awam. Anda bisa langsung buka file, dengarkan hasil, lalu export.

Apakah semua file pasti jadi bagus?

Tidak selalu. File rekaman yang bersih biasanya hasilnya lebih bagus. Kalau file sudah pecah, clipping, noise berat, atau terlalu rusak, hasilnya tetap punya batasan.

Apakah bisa menghilangkan noise berat?

Bukan fokus utama. DeSonKuPik lebih untuk tone, loudness, clarity, stereo, dan finishing. Untuk noise berat, butuh tool restoration khusus.

Apakah audio saya di-upload ke internet?

Untuk aplikasi desktop, workflow normalnya lokal di komputer: buka file dari komputer, proses di aplikasi, lalu export ke komputer.

Bisa untuk video YouTube?

Bisa. Cocok untuk voiceover, audio review, musik demo, atau audio dari video yang ingin dibuat lebih rapi sebelum upload.

Bisa untuk podcast?

Bisa. Aplikasi ini bisa membantu suara bicara lebih terkontrol dan lebih polished, tetapi kualitas mic dan cara rekaman tetap sangat berpengaruh.

Export WAV atau MP3, pilih yang mana?

Pilih WAV jika ingin kualitas tinggi untuk arsip atau editing lanjutan. Pilih MP3 jika ingin file kecil untuk dikirim atau dibagikan.

Kenapa Windows/macOS memberi warning?

Build open-source sering belum ditandatangani sertifikat berbayar. Windows SmartScreen atau macOS Gatekeeper bisa memberi warning karena app belum code-signed.

Boleh dipakai komersial?

Cek license repo dan halaman business-use terlebih dahulu. Untuk redistribusi perusahaan, OEM, white-label, atau produk closed-source, hubungi SonKuPik.
