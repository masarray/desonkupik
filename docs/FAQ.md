# DeSonKuPik FAQ

## What is DeSonKuPik?

DeSonKuPik is an audio mastering app. It helps make an existing audio file sound more polished before you publish or share it.

## What does mastering mean?

Mastering is the final finishing step for audio. It usually improves loudness, tonal balance, clarity, stereo feel, and final peak control.

## Is it a music maker?

No. DeSonKuPik does not create songs from zero. It improves audio files you already have.

## Do I need audio engineering knowledge?

No. Beginners can start by opening a file, previewing the default chain, and exporting the result.

## Will it always make every file better?

No. A clean recording usually works better. If the file is already badly distorted, clipped, or noisy, the result has limits.

## Can it remove noise?

It is not a dedicated restoration tool. It focuses on tone, loudness, clarity, stereo feel, and final polish.

## Is my audio uploaded?

The desktop app is designed for local file open and export workflow.

## Can I use it for YouTube?

Yes. It can help polish voiceover, music demos, reviews, and exported video audio.

## Can I use it for podcasts?

Yes. It can help speech sound more controlled and finished.

## Can I use it for music?

Yes, especially for demos and creator content. For serious commercial music release, compare with reference tracks.

## WAV or MP3?

Use WAV for best quality and editing. Use MP3 for smaller file size and easy sharing.

## Why does Windows show a warning?

Unsigned community builds may trigger SmartScreen because Windows does not yet recognize the publisher certificate.

## Why does macOS show a warning?

Unsigned builds may be blocked by Gatekeeper. You may need to allow the app manually from Privacy & Security settings.

## Is it open source?

Yes, the source is public on GitHub. Follow the repository license terms.

## Can I use it commercially?

Check the repository license and business-use information. For company redistribution, OEM, or closed-source commercial use, contact SonKuPik.
