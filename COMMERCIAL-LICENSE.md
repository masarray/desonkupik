# Commercial Services and Partnership

DeSonKuPik source code is published under the repository source license. This file is not intended to replace or reduce the public source-code license.

For businesses, agencies, integrators, and product teams, SonKuPik may provide separate written commercial arrangements around services and brand use.

Examples:

- OEM integration.
- White-label product development.
- Custom audio engine tuning.
- Custom preset design.
- Priority support.
- Enterprise deployment assistance.
- Official brand partnership.
- Training, consulting, or implementation support.

## When should you contact SonKuPik?

Contact SonKuPik if you want to:

- Distribute a modified build under official SonKuPik or DeSonKuPik branding.
- Bundle the application into a paid product or hardware/software package.
- Use the official logo, icon, or visual identity commercially.
- Build a closed-source product based on project-specific work and need separate written terms.
- Request custom DSP, presets, or enterprise support.

## Normal use

Normal use of the official app and normal open-source use should follow the repository license and documentation.

This page is a practical business-use guide, not legal advice.
